欢迎来到 `www.jarodleo.top`。

这个网站使用 Hexo 生成，部署目标是 GitHub Pages，并通过自定义域名 `www.jarodleo.top` 访问。它会作为一个长期维护的个人空间，用来记录技术笔记、研究想法、项目复盘和生活观察。

## 网站会放什么

- 技术笔记：开发经验、工具链、问题排查、源码阅读。
- 研究记录：论文阅读、实验想法、方法总结。
- 项目复盘：做过的项目、踩过的坑、后续计划。
- 生活随笔：旅行、阅读、效率系统与长期主义。

## 为什么选择 Hexo

Hexo 的优点很直接：文章用 Markdown 管理，构建结果是静态文件，部署到 GitHub Pages 后稳定、免费、速度快。对个人博客来说，它的复杂度刚刚好。

## 下一步

你可以从 `site/source/_posts/` 新建 Markdown 文件开始写文章，也可以先修改 `about`、`projects`、`links` 几个页面，把占位内容替换成真实信息。
